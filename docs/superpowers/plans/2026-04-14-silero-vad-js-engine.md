# SileroVAD Pure-JS Inference Engine Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a pure-JavaScript inference engine for SileroVAD v5 that runs in iOS Safari (where ONNX Runtime WASM is broken since iOS 16.4+), so QuizVidya students on iPhones get streaming voice activity detection instead of hold-and-release.

**Architecture:** A Python toolchain extracts model weights from the official ONNX file into a compact binary blob and produces "golden" reference inputs/outputs from ONNX Runtime. A small JS library implements the layer primitives (Conv1D, LSTM cell, Linear, sigmoid/tanh/ReLU), assembles them into the SileroVAD forward pass, and is validated against the golden fixtures to within `1e-4` absolute error. An AudioWorklet buffers raw mic PCM into 512-sample frames; a `VADRecorder` class owns speech-start/speech-end state machine logic.

**Tech Stack:** Vanilla JS (ESM, no TypeScript build), Vitest (tests), Python 3 + `onnx` + `onnxruntime` + `numpy` (weight extraction & golden generation), Web Audio API + AudioWorklet (browser runtime).

**Open decisions baked in (override before execution if wrong):**
- Model: v5 (`silero_vad.onnx` from the official repo, 16kHz path only)
- Weights serialized as raw little-endian `float32` binary + a small JSON manifest (name → offset, length, shape)
- Test runner: Vitest 4.x
- No bundler — ship ESM modules directly; consumer apps bundle if they want
- Golden tolerance: `abs(js - ort) < 1e-4` per element, per frame
- Repository layout is greenfield (the repo is currently empty except for the spec)

**Post-Task-1 revision (2026-04-14):** The original spec's architecture description was wrong in several ways. See `docs/ARCHITECTURE.md` for the real layer list. Key corrections:
- **STFT preprocessing is required** — a fixed-weight Conv1D (kernel=256, stride=128) plus real/imag magnitude. The spec omitted this.
- **4 encoder Conv1D layers** (not 1–2). Strides [1, 2, 2, 1], all kernel=3 pad=1.
- **LSTM hidden size = 128** (spec said 64). State is packed `(2, N, 128)` = [h; c].
- **Gate order = PyTorch `[i, f, g, o]`** (not ONNX `[i, o, f, c]`).
- **Decoder output layer is a 1×1 Conv1D** (mathematically equivalent to Linear).
- **Weights live as `Constant` nodes inside nested `If` subgraphs**, not as top-level `graph.initializer`. The export script must recurse.

Tasks 2, 4d, and 6 are revised below to reflect these findings. A new **Task 4e (STFT op)** is inserted between 4d and 5.

import sys
import onnx
import numpy as np
from onnx import numpy_helper

def main(path: str) -> None:
    model = onnx.load(path)
    print("=" * 70)
    print("GRAPH INPUTS")
    print("=" * 70)
    for inp in model.graph.input:
        dims = [d.dim_value or d.dim_param for d in inp.type.tensor_type.shape.dim]
        print(f"  {inp.name}: {dims}")

    print("\n" + "=" * 70)
    print("GRAPH OUTPUTS")
    print("=" * 70)
    for out in model.graph.output:
        dims = [d.dim_value or d.dim_param for d in out.type.tensor_type.shape.dim]
        print(f"  {out.name}: {dims}")

    print("\n" + "=" * 70)
    print("INITIALIZERS (weight tensors)")
    print("=" * 70)
    total_floats = 0
    for tensor in model.graph.initializer:
        arr = numpy_helper.to_array(tensor)
        total_floats += arr.size
        print(f"  {tensor.name:50s} shape={list(arr.shape)} dtype={arr.dtype}")
    print(f"\n  TOTAL: {len(model.graph.initializer)} tensors, {total_floats} floats "
          f"(~{total_floats * 4 / 1024 / 1024:.2f} MB as float32)")

    print("\n" + "=" * 70)
    print("NODES (in topological order)")
    print("=" * 70)
    for i, node in enumerate(model.graph.node):
        ins = ", ".join(node.input)
        outs = ", ".join(node.output)
        print(f"  [{i:3d}] {node.op_type:15s} in=[{ins}] -> out=[{outs}]")

    print("\n" + "=" * 70)
    print("SUBGRAPHS (If/Loop bodies)")
    print("=" * 70)
    for node in model.graph.node:
        for attr in node.attribute:
            if attr.type == onnx.AttributeProto.GRAPH:
                print(f"  Node {node.op_type} has subgraph '{attr.name}' with "
                      f"{len(attr.g.node)} nodes")

if __name__ == "__main__":
    main(sys.argv[1] if len(sys.argv) > 1 else "weights/silero_vad_v5.onnx")

```python
"""Export SileroVAD weights to a flat float32 binary + JSON manifest.

Output:
  weights/silero_vad_v5.bin            # raw little-endian float32 concatenation
  weights/silero_vad_v5.manifest.json  # { "tensors": [{name, shape, offset, length}, ...] }

Offsets and lengths are in *float elements* (not bytes). The JS loader multiplies
by 4 to get byte offsets.
"""
import json
import sys
from pathlib import Path
import onnx
import numpy as np
from onnx import numpy_helper

def export(onnx_path: str, bin_path: str, manifest_path: str) -> None:
    model = onnx.load(onnx_path)
    tensors = []
    blobs: list[np.ndarray] = []
    offset = 0

    for tensor in model.graph.initializer:
        arr = numpy_helper.to_array(tensor).astype(np.float32, copy=False)
        flat = np.ascontiguousarray(arr.flatten())
        tensors.append({
            "name": tensor.name,
            "shape": list(arr.shape),
            "offset": offset,
            "length": int(flat.size),
        })
        blobs.append(flat)
        offset += flat.size

    blob = np.concatenate(blobs).astype(np.float32)
    Path(bin_path).write_bytes(blob.tobytes(order="C"))
    Path(manifest_path).write_text(json.dumps({"tensors": tensors}, indent=2))

    # Self-check: re-read and verify byte count and a sample tensor checksum.
    raw = Path(bin_path).read_bytes()
    assert len(raw) == blob.size * 4, f"size mismatch: {len(raw)} vs {blob.size * 4}"
    reread = np.frombuffer(raw, dtype=np.float32)
    assert np.allclose(reread, blob), "round-trip mismatch"

    print(f"Wrote {len(tensors)} tensors, {blob.size} floats "
          f"({len(raw) / 1024 / 1024:.2f} MB) to {bin_path}")
    print(f"Manifest: {manifest_path}")

if __name__ == "__main__":
    onnx_in = sys.argv[1] if len(sys.argv) > 1 else "weights/silero_vad_v5.onnx"
    bin_out = onnx_in.replace(".onnx", ".bin")
    manifest_out = onnx_in.replace(".onnx", ".manifest.json")
    export(onnx_in, bin_out, manifest_out)
```

```python
"""Run ONNX Runtime over deterministic synthetic frames; record inputs + outputs.

Produces tests/fixtures/golden.json:
{
  "frames": [
    {
      "input": [...512 floats...],
      "state_in": [...256 floats... (2*1*128, flat)],
      "speech_prob": 0.0123,
      "state_out": [...256 floats...]
    },
    ...10 frames...
  ],
  "tolerance": 1e-4
}

The frames are designed to exercise both speech-like and silence-like behavior:
- frames 0-2: zeros (silence)
- frames 3-5: low-amplitude white noise (silence-ish)
- frames 6-9: 200Hz sine wave at 0.3 amplitude (speech-band tone)

State is threaded across frames within each block, but reset between blocks
to make individual frames independently reproducible if needed.
"""
import json
from pathlib import Path
import numpy as np
import onnxruntime as ort

MODEL_PATH = "weights/silero_vad_v5.onnx"
OUT_PATH = "tests/fixtures/golden.json"
SR = 16000
WIN = 512

def make_frames() -> list[np.ndarray]:
    rng = np.random.default_rng(seed=42)
    frames = []
    # silence
    for _ in range(3):
        frames.append(np.zeros(WIN, dtype=np.float32))
    # white noise
    for _ in range(3):
        frames.append((rng.standard_normal(WIN) * 0.01).astype(np.float32))
    # sine
    t = np.arange(WIN) / SR
    for k in range(4):
        sine = 0.3 * np.sin(2 * np.pi * 200 * (t + k * WIN / SR))
        frames.append(sine.astype(np.float32))
    return frames

def main() -> None:
    sess = ort.InferenceSession(MODEL_PATH, providers=["CPUExecutionProvider"])
    # Discover state shape from the model rather than hardcoding.
    state_input = next(i for i in sess.get_inputs() if i.name in ("state", "h", "c"))
    state_shape = [d if isinstance(d, int) else 1 for d in state_input.shape]
    state = np.zeros(state_shape, dtype=np.float32)
    if "h" in [i.name for i in sess.get_inputs()]:
        # legacy split-state model
        h = np.zeros((2, 1, 64), dtype=np.float32)
        c = np.zeros((2, 1, 64), dtype=np.float32)
        use_split = True
    else:
        use_split = False

    sr = np.array(SR, dtype=np.int64)
    out = {"frames": [], "tolerance": 1e-4, "sr": SR, "win": WIN}

    for frame in make_frames():
        x = frame.reshape(1, WIN).astype(np.float32)
        if use_split:
            feeds = {"input": x, "sr": sr, "h": h, "c": c}
            outputs = sess.run(None, feeds)
            speech_prob, h, c = outputs[0], outputs[1], outputs[2]
            state_in_flat = np.concatenate([h.flatten(), c.flatten()]).tolist()
            state_out_flat = state_in_flat  # not used in this branch; loader reads h/c separately
        else:
            feeds = {"input": x, "sr": sr, "state": state}
            outputs = sess.run(None, feeds)
            speech_prob, new_state = outputs[0], outputs[1]
            state_in_flat = state.flatten().tolist()
            state_out_flat = new_state.flatten().tolist()
            state = new_state

        out["frames"].append({
            "input": frame.tolist(),
            "state_in": state_in_flat,
            "speech_prob": float(speech_prob.flatten()[0]),
            "state_out": state_out_flat,
        })

    Path(OUT_PATH).parent.mkdir(parents=True, exist_ok=True)
    Path(OUT_PATH).write_text(json.dumps(out))
    sizes = [f["speech_prob"] for f in out["frames"]]
    print(f"Wrote {len(out['frames'])} golden frames to {OUT_PATH}")
    print(f"Speech probabilities: {[f'{p:.4f}' for p in sizes]}")
    print(f"Expected: low for frames 0-5 (silence/noise), higher for frames 6-9 (sine)")

if __name__ == "__main__":
    main()
```

// Element-wise activations. All operate on or return Float32Array.

export function sigmoid(x) {
  return 1 / (1 + Math.exp(-x));
}

export function tanh(x) {
  return Math.tanh(x);
}

/** ReLU in place. Returns the same array for chaining. */
export function relu(arr) {
  for (let i = 0; i < arr.length; i++) {
    if (arr[i] < 0) arr[i] = 0;
  }
  return arr;

/**
 * Matrix multiply: C (aRows x bCols) = A (aRows x aCols) @ B (aCols x bCols).
 * All inputs are row-major Float32Array. Returns a new Float32Array.
 */
export function matmul(a, aRows, aCols, b, bCols) {
  const c = new Float32Array(aRows * bCols);
  for (let i = 0; i < aRows; i++) {
    for (let k = 0; k < aCols; k++) {
      const aik = a[i * aCols + k];
      if (aik === 0) continue;
      for (let j = 0; j < bCols; j++) {
        c[i * bCols + j] += aik * b[k * bCols + j];
      }
    }
  }
  return c;
}

/**
 * Add a per-row bias in place. x has shape (rows, cols) row-major;
 * bias has length `rows`.
 */
export function addBias(x, rows, cols, bias) {
  for (let i = 0; i < rows; i++) {
    const b = bias[i];
    for (let j = 0; j < cols; j++) {
      x[i * cols + j] += b;
    }
  }
  return x;
}
```

Note the loop order is `i, k, j` (not `i, j, k`) — this is ~2× faster because of cache locality on `b` and lets us hoist `a[i*aCols+k]`.

/**
 * 1D convolution. Layout is channels-first: input is (inChannels, inputLength)
 * flattened row-major; weight is (outChannels, inChannels, kernelSize) row-major;
 * bias has length outChannels. Returns (outChannels, outLen).
 *
 * Padding is symmetric ("same"-style numeric padding). For "same" output length
 * with stride 1, pass padding = (kernelSize - 1) / 2.
 */
export function conv1d(input, weight, bias, opts) {
  const { inChannels, outChannels, kernelSize, stride, padding, inputLength } = opts;
  const paddedLen = inputLength + 2 * padding;
  const outLen = Math.floor((paddedLen - kernelSize) / stride) + 1;
  const out = new Float32Array(outChannels * outLen);

  for (let oc = 0; oc < outChannels; oc++) {
    const wBase = oc * inChannels * kernelSize;
    for (let t = 0; t < outLen; t++) {
      let sum = bias[oc];
      const inStart = t * stride - padding;
      for (let ic = 0; ic < inChannels; ic++) {
        const inBase = ic * inputLength;
        const wcBase = wBase + ic * kernelSize;
        for (let k = 0; k < kernelSize; k++) {
          const idx = inStart + k;
          if (idx < 0 || idx >= inputLength) continue; // zero-pad
          sum += input[inBase + idx] * weight[wcBase + k];
        }
      }
      out[oc * outLen + t] = sum;
    }
  }
  return out;
}

```javascript
/**
 * Single-timestep LSTM cell. Computes:
 *   gates = W_ih @ x + b_ih + W_hh @ hPrev + b_hh        // shape (4*hidden,)
 *   (i, f, g, o) = split(gates) reordered per `gateOrder`
 *   c = sigmoid(f) * cPrev + sigmoid(i) * tanh(g)
 *   h = sigmoid(o) * tanh(c)
 *
 * gateOrder controls how the four contiguous hiddenSize-blocks in `gates`
 * map to (i, f, g, o). PyTorch order is ['i','f','g','o']; ONNX is ['i','o','f','c'].
 */
export function lstmCell(x, hPrev, cPrev, params) {
  const { W_ih, W_hh, b_ih, b_hh, inputSize, hiddenSize, gateOrder } = params;
  const gates = new Float32Array(4 * hiddenSize);

  // gates += b_ih + b_hh
  for (let i = 0; i < 4 * hiddenSize; i++) gates[i] = b_ih[i] + b_hh[i];

  // gates += W_ih @ x
  for (let r = 0; r < 4 * hiddenSize; r++) {
    let sum = 0;
    const base = r * inputSize;
    for (let j = 0; j < inputSize; j++) sum += W_ih[base + j] * x[j];
    gates[r] += sum;
  }
  // gates += W_hh @ hPrev
  for (let r = 0; r < 4 * hiddenSize; r++) {
    let sum = 0;
    const base = r * hiddenSize;
    for (let j = 0; j < hiddenSize; j++) sum += W_hh[base + j] * hPrev[j];
    gates[r] += sum;
  }

  // Resolve gate offsets from the order array.
  const offset = { i: 0, f: 0, g: 0, o: 0, c: 0 };
  gateOrder.forEach((name, idx) => { offset[name] = idx * hiddenSize; });
  const gOff = offset.g || offset.c; // 'g' (PyTorch) and 'c' (ONNX) name the same gate

  const h = new Float32Array(hiddenSize);
  const c = new Float32Array(hiddenSize);
  for (let n = 0; n < hiddenSize; n++) {
    const ig = 1 / (1 + Math.exp(-gates[offset.i + n]));
    const fg = 1 / (1 + Math.exp(-gates[offset.f + n]));
    const cg = Math.tanh(gates[gOff + n]);
    const og = 1 / (1 + Math.exp(-gates[offset.o + n]));
    c[n] = fg * cPrev[n] + ig * cg;
    h[n] = og * Math.tanh(c[n]);
  }
  return { h, c };
}
```

/**
 * Build a name -> {shape, data: Float32Array view} map from a single binary blob
 * and its manifest. Views share the underlying ArrayBuffer (no copy).
 *
 * The manifest's `offset` and `length` are in float-elements (×4 for bytes).
 */
export function loadWeightsFromBuffers(arrayBuffer, manifest) {
  const out = {};
  for (const t of manifest.tensors) {
    out[t.name] = {
      shape: t.shape,
      data: new Float32Array(arrayBuffer, t.offset * 4, t.length),
    };
  }
  return out;
}

/**
 * Convenience: fetch both files in parallel and return the loaded weights.
 * Browser-only (uses fetch).
 */
export async function loadWeights(binUrl, manifestUrl) {
  const [binResp, manifestResp] = await Promise.all([fetch(binUrl), fetch(manifestUrl)]);
  if (!binResp.ok) throw new Error(`weights bin: ${binResp.status}`);
  if (!manifestResp.ok) throw new Error(`weights manifest: ${manifestResp.status}`);
  const [arrayBuffer, manifest] = await Promise.all([
    binResp.arrayBuffer(),
    manifestResp.json(),
  ]);
  return loadWeightsFromBuffers(arrayBuffer, manifest);
}

/**
 * AudioWorklet processor: buffers raw mic PCM into 512-sample frames at the
 * AudioContext sample rate (which we set to 16000 in the recorder) and posts
 * each completed frame to the main thread.
 *
 * Important: the AudioWorklet input arrives in 128-sample render quanta, so
 * 512 samples take 4 quanta to fill (≈8ms each at 16kHz → 32ms per frame).
 */
class VADProcessor extends AudioWorkletProcessor {
  constructor() {
    super();
    this.buffer = new Float32Array(512);
    this.bufferIndex = 0;
  }

  process(inputs) {
    const channel = inputs[0]?.[0];
    if (!channel) return true; // no input this quantum; keep the node alive

    for (let i = 0; i < channel.length; i++) {
      this.buffer[this.bufferIndex++] = channel[i];
      if (this.bufferIndex >= 512) {
        // Transfer a copy; the next frame reuses `this.buffer`.
        this.port.postMessage(this.buffer.slice());
        this.bufferIndex = 0;
      }
    }
    return true;
  }
}

registerProcessor('vad-processor', VADProcessor);
```

```javascript
import { SileroVADJS } from './silero_vad.js';
import { loadWeights } from './weights_loader.js';

/**
 * Pure state machine: takes a frame's speech probability, returns whether
 * a speech-start or speech-end event happened. No audio, no Web Audio API,
 * no DOM — fully unit-testable.
 */
export class SpeechStateMachine {
  constructor({ threshold = 0.5, silenceFramesToEnd = 15 } = {}) {
    this.threshold = threshold;
    this.silenceFramesToEnd = silenceFramesToEnd;
    this.speaking = false;
    this.silenceCount = 0;
  }

  feed(prob) {
    if (prob >= this.threshold) {
      this.silenceCount = 0;
      if (!this.speaking) {
        this.speaking = true;
        return { event: 'speech-start' };
      }
      return { event: null };
    }
    if (this.speaking) {
      this.silenceCount++;
      if (this.silenceCount >= this.silenceFramesToEnd) {
        this.speaking = false;
        this.silenceCount = 0;
        return { event: 'speech-end' };
      }
    }
    return { event: null };
  }

  reset() {
    this.speaking = false;
    this.silenceCount = 0;
  }
}

/**
 * Browser-only: wires getUserMedia → AudioContext(16000) → AudioWorklet → VAD.
 * Emits 'speech-start' and 'speech-end' on EventTarget.
 */
export class VADRecorder extends EventTarget {
  constructor({
    weightsBinUrl,
    weightsManifestUrl,
    workletUrl,
    threshold = 0.5,
    silenceMs = 480, // 480ms ≈ 15 frames at 32ms
  } = {}) {
    super();
    this.weightsBinUrl = weightsBinUrl;
    this.weightsManifestUrl = weightsManifestUrl;
    this.workletUrl = workletUrl;
    const silenceFramesToEnd = Math.round(silenceMs / 32);
    this.stateMachine = new SpeechStateMachine({ threshold, silenceFramesToEnd });
    this.vad = null;
    this.audioContext = null;
    this.stream = null;
    this.audioChunks = [];
  }

  async start() {
    const weights = await loadWeights(this.weightsBinUrl, this.weightsManifestUrl);
    this.vad = new SileroVADJS(weights);

    this.stream = await navigator.mediaDevices.getUserMedia({
      audio: { sampleRate: 16000, channelCount: 1, echoCancellation: true, noiseSuppression: true },
    });
    this.audioContext = new AudioContext({ sampleRate: 16000 });
    await this.audioContext.audioWorklet.addModule(this.workletUrl);

    const source = this.audioContext.createMediaStreamSource(this.stream);
    const node = new AudioWorkletNode(this.audioContext, 'vad-processor');
    node.port.onmessage = (e) => this._onFrame(e.data);
    source.connect(node);
    // Don't connect node to destination — we don't want to play mic back.
  }

  _onFrame(frame) {
    const prob = this.vad.process(frame);
    const { event } = this.stateMachine.feed(prob);
    if (event === 'speech-start') {
      this.audioChunks = [frame];
      this.dispatchEvent(new CustomEvent('speech-start'));
    } else if (this.stateMachine.speaking) {
      this.audioChunks.push(frame);
    } else if (event === 'speech-end') {
      this.dispatchEvent(new CustomEvent('speech-end', {
        detail: { audio: concatFloat32(this.audioChunks), sampleRate: 16000 },
      }));
      this.audioChunks = [];
      this.vad.reset();
    }
  }

  async stop() {
    if (this.audioContext) await this.audioContext.close();
    if (this.stream) this.stream.getTracks().forEach((t) => t.stop());
    this.audioContext = null;
    this.stream = null;
  }
}

function concatFloat32(chunks) {
  let total = 0;
  for (const c of chunks) total += c.length;
  const out = new Float32Array(total);
  let o = 0;
  for (const c of chunks) { out.set(c, o); o += c.length; }
  return out;
}
```

        ├── weights_loader.js

        └── index.js

```bash

### nginx checklist

1. **`.js` MIME type** — confirm with `curl -I https://yourdomain/static/vad/vad_processor.js`; `Content-Type` must be `application/javascript` or `text/javascript`. Default `mime.types` covers this.

2. **Long cache on the weights** — add a `location` block:

   ```nginx
   location ~* /static/vad/.*\.(bin|json)$ {
       expires 1y;
       add_header Cache-Control "public, immutable";
   }
   ```

   When you upgrade the model, version with a query string in the template (`silero_vad_v5.bin?v=2`) — don't rename the file.

3. **Same-origin for the worklet** — AudioWorklet refuses cross-origin module loads even with permissive CORS headers. Serving from `/static/vad/` (same origin as the page) is the simplest fix. If you must put assets on a CDN, the `.bin` and `.json` can live there, but `vad_processor.js` must come from the Flask origin.

### Flask gotcha

If you serve Flask behind nginx via `proxy_pass`, make sure nginx isn't rewriting `Content-Type` on `.js` responses. The default `proxy_pass` preserves headers from upstream — but if you have `proxy_set_header` rules or a `sub_filter` MIME-rewriting block, AudioWorklet will silently fail to load.

## Risks / Things That Might Bite You

1. **Tensor name aliases:** ONNX exports prefix names like `_model.lstm.weight_ih_l0`. The `_name()` map in Task 6 must be filled in from the actual `inspect_model.py` output. Symptom: `Cannot read properties of undefined (reading 'shape')` in `_lstmHiddenSize`.
2. **LSTM gate ordering:** Tested both orderings (PyTorch `[i,f,g,o]` vs ONNX `[i,o,f,c]`). Symptom: probabilities are roughly correct but state diverges over multiple frames.
3. **STFT/FFT layer:** If Netron reveals the v5 model has an FFT op (it shouldn't — the encoder is conv-based — but legacy did), you'll need to implement a small STFT in `ops.js`. Plan for an extra half-day.
4. **iOS getUserMedia HTTPS requirement:** Local self-signed certs need to be trusted in iOS Settings → General → About → Certificate Trust Settings. ngrok or a real HTTPS tunnel is faster.
5. **Float64 vs Float32 precision drift:** Should not exceed `1e-5` over 10 frames. If your golden test fails by `~1e-3` consistently, you have an algorithmic bug, not a precision bug.
