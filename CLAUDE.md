# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What This Is

Pure-JavaScript SileroVAD v5 inference engine — no WASM, no ONNX Runtime. Exists because `onnxruntime-web` WASM is broken on iOS Safari since iOS 16.4+. Reimplements the 16kHz forward pass in vanilla JS so VAD works on all browsers with AudioWorklet support.

## Commands

```bash
npm test                # vitest run — 28 tests across 4 suites
npm run test:watch      # vitest in watch mode (TDD)
npm run demo            # python3 -m http.server 8000, then open examples/index.html
npm run weights:export  # extract 16kHz weights from .onnx → .bin + .manifest.json
npm run weights:golden  # regenerate golden fixtures via official Python wrapper
```

One-time setup (weights must exist before tests pass):
```bash
npm install
pip install -r scripts/requirements.txt
curl -L -o weights/silero_vad_v5.onnx https://github.com/snakers4/silero-vad/raw/master/src/silero_vad/data/silero_vad.onnx
npm run weights:export
npm run weights:golden
```

## Architecture

**Runtime has zero npm dependencies.** Only devDependency is vitest.

### Inference Pipeline (src/)

`silero_vad.js` — core forward pass, 512 samples/frame at 16kHz:
1. **STFT assembly:** prepend 64 context samples from previous frame, append 512 new, right-reflect 64 → 640 samples
2. **STFT Conv1d:** fixed Fourier kernel (256 filters) → 129 freq bins × 4 time steps → magnitude
3. **Encoder:** 4 Conv1d→ReLU layers (strided downsampling) → 128-dim feature
4. **LSTM:** single-step with h/c state carry across frames
5. **Decoder:** ReLU → 1×1 Conv + bias → sigmoid → speech probability [0,1]

`ops.js` — pure-function neural net primitives (matmul, conv1d, lstm_cell, relu, sigmoid, etc.)

`weights_loader.js` — loads `.bin` + `.manifest.json`, returns Float32Array views into the binary ArrayBuffer (no copying)

### Browser Integration (src/)

`vad_recorder.js` — `VADRecorder` (EventTarget) wraps getUserMedia + AudioContext(16kHz) + AudioWorklet. `SpeechStateMachine` converts per-frame probabilities into speech-start/speech-end events with configurable threshold and silence duration.

`vad_processor.js` — AudioWorklet that buffers 128-sample browser quanta into 512-sample frames. Must be served same-origin (AudioWorklet rejects cross-origin loads even with CORS).

### Scripts (scripts/)

Python tools for weight export and fixture generation. Not part of the runtime.

## Key Conventions

- **ES modules throughout** — `"type": "module"` in package.json, all imports use ES import syntax
- **No TypeScript** — vanilla JS with JSDoc comments on public APIs
- **16kHz only** — the 8kHz path in the ONNX graph is intentionally ignored
- **Weight manifest uses float element counts**, not byte offsets — the loader multiplies by 4 for Float32
- **LSTM weights are in PyTorch order** `[i, f, g, o]`, not ONNX order — JS reads directly without reordering
- **Numeric precision:** accumulation in f64 locals, stored to f32; LSTM state kept as f64 Arrays internally. Forward pass matches ORT within 1e-4 per frame

## Testing

Tests live in `tests/*.test.js`. The critical validation is `silero_vad.test.js` which compares frame-by-frame output against golden fixtures (generated by the official Python `silero-vad` wrapper) within 1e-4 tolerance. If you change any op or the forward pass, this test catches numerical regressions.

`vad_recorder.test.js` tests the `SpeechStateMachine` state transitions (not the AudioWorklet, which requires a browser).
